Securing macOS with SleepWatcher


Normally macOS leaves its network connected even while asleep. This allows for push notifications and background processes to continue, such as network date and time. For me, I chose to disable all connections on sleep, partly for safety and also because macOS doesn't always search for previously connected networks. For this we will be using a program called SleepWatcher.


To install SleepWatcher the easy way, I used HomeBrew, a package manager for macOS


Once HomeBrew is sucessfully installed, run this to install SleepWatcher



Now SleepWatcher is installed, we must configure it to turn off wifi(en0) when going to sleep. To do this, create a file in your user directory called .sleep by running "touch ~/.sleep" in Terminal without quotes.



Save it by hitting "ctrl+x" followed by "y" without quotes. Do the same to create a file called ~/.wakeup but replace off with on.




Finally, run both these commands for SleepWatcher to load your wake and sleep scripts.
